## Prerequisites

* Linux, Macos, WSL
* PHP >=8.3
* Xdebug
* Make
* Git

## Setup

```bash
git clone git@github.com:Airat29/php-project-45.git
cd php-project-45 
make install
```

## Play

To play the game use 'make' and the name of the desired game.

Example:
```sh
make brain-prime
```
